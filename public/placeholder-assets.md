# Required Assets

This portfolio requires the following 3D models and images. You'll need to download them separately:

## 3D Models (place in `/public/` directory):

1. **Desktop PC Model**: `/public/desktop_pc/scene.gltf`
   - Download a 3D computer/desktop model in GLTF format
   - Place it in `public/desktop_pc/` folder

2. **Planet Model** (optional): `/public/planet/scene.gltf`
   - Download a 3D planet/Earth model in GLTF format
   - Place it in `public/planet/` folder
